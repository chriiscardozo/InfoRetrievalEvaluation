# InfoRetrievalEvaluation
Evaluation project for IR system at: https://github.com/chriiscardozo/InfoRetrievalCysticFibrosis

Para executar o programa:
	python3 main.py dir_v1 dir_v2
      
	- dir_v1: diretório de arquivos para entrada e saída da versão 1 do buscador (ex.: output_original)
	- dir_v2: diretório de arquivos para entrada e saída da versão 2 do buscador (ex.: output_stem)

Comando de exemplo:
	python3 main.py output_original/ output_stem/
