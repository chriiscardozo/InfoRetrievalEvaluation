# InfoRetrievalEvaluation
Evaluation project for IR system at: https://github.com/chriiscardozo/InfoRetrievalCysticFibrosis

Para executar o programa:
	python3 main.py diretorio
      
	- diretorio: caminho para pasta onde estarão presentes os arquivos output_busca.csv e output_pc_esperado.csv. Também será a pasta de saída do programa para gráficos.
